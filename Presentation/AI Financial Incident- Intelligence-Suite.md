

# 📦 **AI Financial Incidents Intelligence Suite**

<br><br>

## 🧭 Overview

This project integrates multiple layers of an AI-driven incident analysis system for the financial sector, including:

* 📊 Interactive data visualization (Streamlit)
* 🔌 RESTful API for structured access (Render Cloud)
* ⚛️ Multilingual presentation layer (React)
* 📑 Academic/strategic slides (PowerPoint)
